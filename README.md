# deal-ai-finder

**Deal AI Finder** — Buscador de ofertas imperdibles en tiendas chilenas 🇨🇱

Encuentra bugs de precio, descuentos extremos y ofertas increíbles. Todos los precios se muestran en **pesos chilenos (CLP)**.

## Características

- 🔍 Búsqueda en tiempo real en MercadoLibre Chile
- 🚨 Detección automática de bugs de precio (descuentos 80%+)
- 🔥 Clasificación de ofertas: bug → increíble → gran descuento → buen precio
- 💸 Filtros por precio máximo y descuento mínimo
- 🇨🇱 Resultados en pesos chilenos (CLP)
- 🚚 Indicador de envío gratis

## Stack

- [Next.js 16](https://nextjs.org/) + TypeScript
- [Tailwind CSS v4](https://tailwindcss.com/)
- [MercadoLibre Chile API](https://developers.mercadolibre.com.ar/) (pública, sin autenticación)

## Desarrollo

```bash
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador.
